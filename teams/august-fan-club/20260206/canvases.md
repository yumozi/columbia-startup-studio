# Lean Canvas 2: The "Course Hub" App

| **1. Problem** | **2. Solution** | **3. Unique Value Proposition** |
| :--- | :--- | :--- |
| **Top 3 Problems:**<br>1. **Digital Fragmentation:** Materials scattered (Drive, Slack, LMS).<br>2. **High Cognitive Load:** Energy wasted finding files.<br>3. **Context Switching:** Jumping apps breaks focus.<br><br>**Existing Alternatives:**<br>* Notion (Too complex/blank slate)<br>* Google Drive (No context)<br>* Physical Planners (Don't sync)<br>* University LMS (Clunky) | **Top Features:**<br>1. **Course Dashboard:** Pre-structured home for each class.<br>2. **Unified "Pins":** Syllabus + Links + Notes side-by-side.<br>3. **Integrated Tools:** Built-in todo list & note-taker.<br>4. **Global Search:** Find anything instantly. | **Single Clear Message:**<br>"Stop searching. Start studying."<br><br>**High-Level Concept:**<br>The "digital three-ring binder" for the modern student. |
| **4. Key Metrics** | **5. Unfair Advantage** | **6. Channels** |
| * **# of Courses Created** (Activation)<br>* **Weekly Active Users** (Study session frequency)<br>* **Items Pinned per Course**<br>* **Premium Conversion Rate** | * **"Opinionated" Design:** No setup required. The structure *is* the feature.<br>* **Niche Focus:** 100% built for students; ignores other verticals. | * **TikTok/Reels:** "Studygram" influencers.<br>* **ASO:** Keywords like "homework planner."<br>* **Campus Ambassadors:** Word-of-mouth. |
